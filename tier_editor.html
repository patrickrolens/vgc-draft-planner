Reset to Default?
All current tier assignments will be discarded and replaced with the values from pokemon.json. The tier list will also reset to 0–21. This cannot be undone.
Make Current Tiers Default?
The current tier assignments will be saved as the new baseline for this session. "Reset to Default" will restore to this state going forward. (This does not modify pokemon.json - use Export CSV + update_pokemon.py for that.)
